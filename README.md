# Wiigos PWA

Progressive Web App с интерактивными изображениями.

## Деплой

Приложение автоматически деплоится на GitHub Pages при пуше в main/master ветку.

## Локальная разработка

Откройте `index.html` в браузере или используйте простой HTTP сервер:

```bash
python3 -m http.server 8000
```

Затем откройте http://localhost:8000
